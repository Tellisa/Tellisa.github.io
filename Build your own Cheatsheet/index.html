HTML Semantic Elements Cheatsheet

Tag

Name

Description

<header>

Header

This is used for navigational links as well as introductory content ranging from <h1> to <h6>.

<nav>

Nav

This is used to define a block of navigation links like menus and tables of contents.

<main>

Main

This is used to encapsulate the dominant content within a webpage.

<footer>

Footer

It contains information like contact information, copyright information ,terms of use and site map.

<article>

Article

This holds content that makes sense on its own. It can hold content like blogs, comments and magazines.

<section>

Section

This defines elements in a document like chapters and headings.

<aside>

Aside

This is used to mark additional information that can enhance another element but isn't required in order to understand the main content.

<figure>

Figure

This is used to encapsulate media such as an image, illustration, diagram or code snippet , which is referenced in the main flow of the document.

<figcaption>

Figcaption

This is used to descibe the media in the figure tag.

<audio>

Audio

This is used to embed audio content into a document.

<video>

Video

This allows us to add videos to websites.

<embed>

Embed

This is used to incorporate any media content into a page.
